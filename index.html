The Gallery: Selected Works

In the 3D gallery room, you can walk through an exhibition of my projects. Here is a breakdown of some of my work.

Fake Medicine Detector System

An AI-based fake medicine detection system that uses image-based medicine verification with deep learning, built with PyTorch, FastAPI, Streamlit, Grad-CAM and Docker.

Document QA RAG Assistant

An AI-powered document Q&A assistant for PDF-based question answering using Retrieval-Augmented Generation, built with LangChain, FAISS, the OpenAI API and Streamlit.

Deepfake / Fake Audio Detection System

An Android + AI/ML system for detecting deepfake or fake audio using audio feature extraction and model inference, built with Kotlin, FastAPI, Librosa, MFCC and TensorFlow Lite.

n8n Automation Workflows

A set of automation workflows including client order-to-invoice automation, an RSS-to-email news digest, a Gmail inbox logger, and a daily WhatsApp weather report — built with n8n, Airtable, Twilio and Google Sheets.
